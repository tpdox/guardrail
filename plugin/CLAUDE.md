## Setup

1. Create a config file at `~/.config/guardrail/guardrail.yml` (copy `guardrail.yml.example` from the repo and fill in your values)
2. The plugin auto-installs the `guardrail-mcp` binary via `uv tool install` on first use

## Configuration

The config file is searched in this order:
1. `$GUARDRAIL_CONFIG` environment variable
2. `~/.config/guardrail/guardrail.yml`
3. `guardrail.yml` in the current working directory

## Tools

| Tool | Description |
|------|-------------|
| `guardrail_status` | Project state: manifest age, changed models, blast radius |
| `guardrail_review` | Full review pipeline: generate checks, execute SQL, evaluate results |
| `guardrail_checks` | Preview generated SQL checks without executing |
| `guardrail_dashboard` | HTML dashboard from last review results |

## Slash Command

Use `/guardrail` to run a review interactively.
